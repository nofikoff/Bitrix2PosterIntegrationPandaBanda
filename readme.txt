Интеграция заказов из магазина на Bitrix с CRM кафе https://dev.joinposter.com/
Служба доставки ПандаБанда

- Скрипт интегрирован в модуль магазина Битрикс
- Перехватывает заказы и на базе правил соответсвия товаров БИТРИКС - ПОСТЕРУ в базе данных и других правил, формирует API запрос в CRM
- В случае ошибки соответсвия товаров БИТРИКС - ПОСТЕРУ, информирует в телеграм разработчика

Самое сложное было разобраться с обьектом заказов БИТРИКСА и не совершенным API ПОСТЕРА
На каждом этапе все запросы логируются
